# Key Bank

Sito web semplice per Key Bank con pagina di login.

## Struttura

- `index.html` - Pagina principale di login
- `styles.css` - Stili CSS per il design moderno
- `script.js` - JavaScript per la gestione del form
- `icon.ico` - Favicon (da caricare)

- Pagina di login responsive e moderna
- Design professionale con gradiente viola/blu

## Utilizzo

Apri semplicemente `index.html` nel browser per visualizzare la pagina di login.

**Nota:** Ricordati di aggiungere il file `icon.ico` nella root del progetto per la favicon.
